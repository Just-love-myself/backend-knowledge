## 전술적 설계 (Tactical Design)

전략적 설계, 보편 언어를 통해 도메인 모델을 발견했다고 가정했을 때, \
이를 그대로 설계, 코드에 반영하는 걸 **모델 주도 설계**라고 한다.

제일 코드랑 맞닿아 있고, 전략적 설계가 아닌 전술적 설계만 적용하면 DDD Lite라고 부르기도 한다.

도메인 모델은 몇 가지로 구분할 수 있는데, 대표적인 패턴은 아래와 같다.

1. Entity
2. Value Object
3. Aggregate
4. Repository
